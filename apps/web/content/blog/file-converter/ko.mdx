## 자주 묻는 질문

### HWPX 파일이란 무엇인가요?

HWPX는 한컴에서 만든 XML 기반 문서 형식으로, 한국의 대표적인 오피스 소프트웨어인 한컴오피스의 표준 형식입니다. 한국 정부 양식, 대학 문서, 법률 계약서, 사업 서류에서 표준으로 사용됩니다. 기존의 바이너리 HWP 형식을 대체한 새로운 형식입니다.

### 한컴오피스 없이 HWPX를 변환할 수 있나요?

네. Xenith는 클라우드에서 HWPX XML 구조를 직접 파싱합니다. 데스크톱 소프트웨어가 필요하지 않습니다. 브라우저를 통해 파일을 업로드하면 몇 초 안에 PDF 또는 텍스트 파일을 받을 수 있습니다. 설치, 플러그인, 한국 소프트웨어가 일절 필요 없습니다.

### 변환기는 무료인가요?

네, 완전 무료입니다. 회원가입 불필요, 워터마크 없음, 일일 사용 제한 없음. 파일 변환은 운영 비용이 낮아 한국 문서의 접근성을 높이기 위한 미션의 일환으로 무료 제공합니다.

### Xenith는 제 파일을 저장하나요?

아닙니다. 모든 파일은 메모리에서 처리되며 변환 완료 후 자동 삭제됩니다. 디스크에 기록되지 않습니다. 아무것도 보관되지 않습니다. 문서의 프라이버시가 보장됩니다.

### OCR에서 어떤 이미지 형식을 지원하나요?

PNG, JPG, JPEG, WebP, BMP, TIFF를 지원합니다. OCR 엔진(Mistral OCR)은 한국어, 일본어, 중국어, 영어를 포함한 다국어 텍스트를 정확하게 처리합니다. 여러 언어가 섞인 문서도 인식 가능합니다.

### 파일 크기 제한이 있나요?

업로드당 최대 20MB까지 지원합니다. 일일 사용 제한은 없으며 변환 횟수에도 제한이 없습니다.
